# mvc framework php

Aprendiendo cakephp, codeigniter, laravel, symfony, yii2

## Usar un servidor Web

* Laravel ->
      php artisan serve
* Symfony ->
      php bin/console server:run
* Yii2 ->
      php yii serve
* CodeIgniter ->
      php -S localhost:8000
* CakePHP ->
      php bin/cake.php server
